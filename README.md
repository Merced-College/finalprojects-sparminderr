// Parminder Singh
// Text Adventure Game 
    - This is a text based adventure game where your goal is to gather 3 items and win. You travel to different places and fight monsters for loot. There's 3 items you need to collect the Silver Ring, Wolf Fang, and  the Ancient Coin. Once collected, talk to the guard in the town to win. 

// How To Run
    - Run in Main.java
    - Follow the prompts 

// Features
    - Explore the world (multiple locations)
    - Turn based combat (Random attack damage value)
    - Collect items to win
    - Inventory system 

// Data Structures Used
    - ArrayList to store inventory items 
    - HashMap to store weapon damage values for different weapons
    - Various objects such as player, monster, inventory classes 

// Algorithms Used
    - Combat Algorith, turn based fight system 
    - Linear Search, checking win conditions
    - Sorting algorithm, to list the items 
    - Game Navigation, different branches the player can choose.
